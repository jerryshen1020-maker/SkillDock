## 0. 规范优先级

当文档之间存在冲突，按以下优先级执行：
1. `需求总结.md`（业务语义与范围边界）
2. `02-全状态设计参考.html`（状态与交互规则）
3. `01-设计稿.html`（视觉与静态布局）

本轮明确约束：
- 技能操作语义为“移除 / 清空全部”，不再使用 Toggle。
- 应用切换语义为 click，不再使用 hover 自动切换。
- 设置页收敛，不保留主题模式配置入口。

## 5. Skills 页面规范

### 5.1 顶部区

```
Claude Code - Skills   共 N 个      [🗑️ 清空全部] [↻ 一键同步]
```

- 标题：`18px / 600`
- 数量：`14px / 500 / #adb5bd`
- 操作按钮：
  - 清空全部：warning 风格（浅黄底）
  - 一键同步：primary 风格（蓝底白字）

### 5.2 清空全部显隐规则

- 非空列表：显示“清空全部”
- 空状态：隐藏“清空全部”

### 5.3 卡片规范（移除 Toggle）

卡片结构：
```
┌─────────────────────────────────────┐
│ [icon] skill-name           [移除]  │
│ 来源: xxx                               │
│ 描述文案                                 │
│ [AppTag]                                │
└─────────────────────────────────────┘
```

- 卡片背景：`#ffffff`
- 边框：`1px #f1f3f5`
- 圆角：`14px`
- padding：`16px`
- hover：边框变 `#4c6ef5` + `shadow-md` + `translateY(-2px)`
- 移除按钮：
  - 默认：透明背景 + `#adb5bd`
  - hover：`#fff5f5` + `#fa5252`
- 图标容器：`36x36`，背景 `#e7f5ff`，圆角 `10px`

### 5.4 空状态

- 主文案：`暂无已安装的 Skill`
- 副文案：`点击「一键同步」从来源目录安装 Skill 到全局`
- 图标：`48px`，`opacity 0.5`
- 仅保留一个主操作按钮：“一键同步”

### 5.5 加载态（切应用）

- 文案：`正在加载已安装的 Skill...`
- 使用 spinner + 文案的居中块
- 加载期间禁止误触导致的重复切换

### 5.6 同步中态

- 同步按钮变为禁用态
- 按钮文案：`同步中...`
- 可显示区域级提示：`正在扫描来源目录并同步...`

## 6. 来源管理页规范

### 6.1 页面结构

1) 本地目录模块  
2) 分割线  
3) Git 仓库模块

- 本地区操作：`+ 添加目录`
- Git 区操作：`↻ 一键更新` + `+ 添加仓库`

### 6.2 来源行（Source Item）

- 背景：白色
- 边框：`1px #f1f3f5`
- 圆角：`10px`
- 内边距：`12px`
- 删除按钮默认隐藏，hover 时显示

### 6.3 Git 进度展示（必须在 Git 模块内）

- 位置：Git 仓库标题下、列表上方
- 样式：`#e7f5ff` 背景，蓝色文案，14px spinner
- 文案示例：`正在克隆 skills-hub 仓库... 45%`
- 行为：后台执行，不弹阻塞弹窗

### 6.4 来源错误态

- 页面显示 error banner（浅红底红字）
- 失败来源项保留在列表，允许删除或重试
- 错误来源可用红色边框强调

## 9. 全状态清单（必须覆盖）

1. 正常态：有已安装技能  
2. 空态：无已安装技能  
3. 加载态：切换应用后读取目录  
4. 同步中态：一键同步执行中  
5. 来源管理空态：无本地目录/无 Git 来源  
6. 来源管理 Git 克隆中：显示进度且页面可操作  
7. 来源管理扫描失败：错误提示 + 可恢复入口  
8. Toast 成功/警告/失败  
9. 移除确认弹窗  
10. 清空全部确认弹窗

## 10. 交互行为规则（实现约束）

### 10.1 Skills 列表

- 点击侧边栏应用：读取目标目录并展示该应用已安装技能
- 点击一键同步：扫描来源并同步到目标目录；按钮进入同步中状态
- 点击移除：二次确认后删除当前应用目标目录中的对应技能项
- 点击清空全部：二次确认后删除目标目录全部子项并进入空态
- 搜索关键词：仅过滤当前应用已安装列表（name/description）

### 10.2 来源管理

- 添加目录：系统目录选择器（NSOpenPanel）
- 添加仓库：后台 clone，来源列表显示进度
- 一键更新：后台 pull 全部 Git 来源并展示过程反馈
- 来源扫描失败：保留失败项，可重试或删除
