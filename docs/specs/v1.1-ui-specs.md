## 2. 侧边栏 (Sidebar)

### 2.1 SidebarView 组件

**文件位置**：`SkillDock/Views/Sidebar/SidebarView.swift`

**结构**：
```
┌──────────────┐
│ [浏览]       │  ← SidebarHeader
│              │
│ 📦 Skills仓库│  ← SidebarEntry (active)
│ ⭐ 常用项目  │  ← SidebarEntry
│ 📋 全部项目  │  ← SidebarEntry
│              │
│ ─────────    │
│              │
│ ⚙️ 设置      │  ← SidebarEntry (bottom)
└──────────────┘
```

### 2.2 SidebarEntryView 组件规范

| 状态 | 背景 | 文字色 | Badge 背景 | Badge 文字色 |
|------|------|--------|-----------|-------------|
| Default | `#f1f3f5` | `#868e96` | `#f1f3f5` | `#adb5bd` |
| Hover | `#e9ecef` | `#212529` | - | - |
| Active | `#e7f5ff` | `#4c6ef5` | `#4c6ef5` | `#ffffff` |

**尺寸**：
- `padding`：`10px` horizontal, `10px` vertical
- `iconSize`：`16px`
- `fontSize`：`13px`
- `cornerRadius`：`10px`
- `marginBottom`：`2px`

**代码约束**：
```swift
struct SidebarEntryView: View {
    let icon: String
    let title: String
    let badge: Int?
    let isActive: Bool

    var body: some View {
        HStack(spacing: 8) {
            Text(icon).font(.system(size: 16))
            Text(title).font(.system(size: 13))
            if let badge = badge {
                Text("\(badge)")
                    .font(.system(size: 11))
                    .padding(.horizontal, 6)
                    .padding(.vertical, 2)
                    .background(isActive ? Color(hex: "#4c6ef5") : Color(hex: "#f1f3f5"))
                    .foregroundColor(isActive ? .white : Color(hex: "#adb5bd"))
                    .cornerRadius(10)
            }
            Spacer()
        }
        .padding(10)
        .background(isActive ? Color(hex: "#e7f5ff") : Color(hex: "#f1f3f5"))
        .foregroundColor(isActive ? Color(hex: "#4c6ef5") : Color(hex: "#868e96"))
        .cornerRadius(10)
    }
}
```

### 3.7 代码约束模板

```swift
struct SkillCardView: View {
    let skill: Skill
    let isEnabled: Bool
    let onToggle: (Bool) -> Void
    let onDetails: () -> Void

    var body: some View {
        VStack(alignment: .leading, spacing: 16) {
            // Header
            HStack {
                // Icon + Info
                HStack(spacing: 8) {
                    RoundedRectangle(cornerRadius: 10)
                        .fill(Color(hex: "#e7f5ff"))
                        .frame(width: 36, height: 36)
                        .overlay(Text(skill.icon).font(.system(size: 18)))

                    VStack(alignment: .leading, spacing: 2) {
                        Text(skill.name)
                            .font(.system(size: 15, weight: .semibold))
                            .foregroundColor(Color(hex: "#212529"))
                            .lineLimit(1)

                        Text(skill.sourcePath)
                            .font(.system(size: 11))
                            .foregroundColor(Color(hex: "#adb5bd"))
                            .lineLimit(1)
                    }
                }
                Spacer()
                // Toggle
                Toggle("", isOn: Binding(...))
                    .labelsHidden()
            }

            // Description
            Text(skill.description)
                .font(.system(size: 13))
                .foregroundColor(Color(hex: "#868e96"))
                .lineLimit(2)

            // Footer
            HStack {
                // Tags
                ForEach(skill.tags, id: \.self) { tag in
                    Text(tag)
                        .font(.system(size: 11))
                        .padding(.horizontal, 8)
                        .padding(.vertical, 3)
                        .background(Color(hex: "#f1f3f5"))
                        .foregroundColor(Color(hex: "#adb5bd"))
                        .cornerRadius(12)
                }
                Spacer()
                // Details Button
                Button("详情") { onDetails() }
                    .font(.system(size: 12))
                    .foregroundColor(Color(hex: "#4c6ef5"))
            }
        }
        .padding(16)
        .background(Color(hex: "#ffffff"))
        .cornerRadius(14)
    }
}
```

## 6. 空状态 (Empty States)

### 6.1 规范

| 场景 | 图标 | 主文案 | 副文案 |
|------|------|--------|--------|
| 无来源 | 📁 | 暂无来源目录 | 点击「添加来源」开始... |
| 无 Skills | 🧩 | 该目录下没有找到 Skill | 请确认目录包含有效的 SKILL.md |
| 搜索无结果 | 🔍 | 未找到匹配的 Skill | 试试其他关键词 |
| 筛选无结果 | 📭 | 该来源下暂无 Skill | - |

**样式**：
- `iconSize`：`48px`
- `opacity`：`0.5`
- `padding`：`48px`

### 6.2 加载态 (Loading State)

| 属性 | 值 |
|------|-----|
| 图标 | ⏳ |
| 图标大小 | 48px |
| 透明度 | 0.5 |
| 主文案 | "正在加载 Skills..." |
| 副文案 | "已发现 X 个 Skills" (可选，扫描中显示) |

**触发场景**：
- 首次加载
- 刷新/同步时

### 6.3 错误态 (Error States)

| 属性 | 值 |

|------|-----|

**结构**：警告横幅 + 来源状态卡片

## 7. 交互行为规则表

| 触发 | 用户操作 | 系统行为 | 反馈 |
|------|----------|----------|------|
| 用户 | 点击「一键同步」 | 扫描所有来源目录，比对新增/删除/变更 | Toast: 同步中 → 同步完成/失败 |
| 用户 | 点击「添加来源」 | 弹出系统目录选择器 | 选择后验证目录有效性 |
| 用户 | 点击 Skill Toggle | 更新配置状态，写入 .claude/settings.json | Toast: 已启用/已禁用 |
| 用户 | 输入搜索关键词 | 实时过滤 Skills 列表（名称/描述） | 无结果时显示空态 |
| 用户 | 点击来源筛选标签 | 过滤指定来源的 Skills | 无结果时显示空态 |
| 用户 | 点击侧边栏入口 | 切换内容区视图 | 高亮当前选中项 |
| 用户 | 点击「添加项目」 | 弹出目录选择器，选择项目目录 | 验证后添加并扫描 |
| 系统 | 启动时 | 加载配置，扫描来源目录 | 显示加载态 → 正常态/错误态 |
| 系统 | 检测到目录不存在 | 标记该来源为不可用 | Toast 警告 + UI 提示 |
| 系统 | 切换项目 | 加载项目级配置，更新 Skills 状态 | Skills 列表反映项目配置 |

## 9. 常见错误 ⚠️

| 错误 | 正确做法 |
|------|----------|
| 硬编码颜色值 | 使用本规范文档中定义的颜色值 |
| 随意圆角 | 使用本规范文档中定义的圆角值 |
| 随意间距 | 使用本规范文档中定义的间距值 |
| 忽略 Hover 态 | 所有可交互元素需定义 Hover 态 |
| Toggle 尺寸错误 | 必须 44x24 |
| 缺少状态设计 | 确保覆盖正常/加载/空/错误/部分可用五种状态 |
