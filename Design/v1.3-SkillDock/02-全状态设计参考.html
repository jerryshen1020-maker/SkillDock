10. 交互行为规则表

技能列表操作

触发 操作 系统行为 反馈
用户 点击侧边栏应用 读取目标应用 ~/.xxx/skills/ 目录，列出已安装的 skill 切换到对应应用的 skill 列表，加载态后展示
用户 点击「一键同步」 扫描所有来源目录，计算差异，创建/更新/删除符号链接 按钮显示同步中，完成后 Toast 通知
用户 点击「移除」 删除当前应用目录下该 skill 的符号链接 显示确认弹窗，确认后 Toast 通知
用户 点击「清空全部」 遍历目标目录，删除所有子项（不管来源） 显示确认弹窗，确认后 Toast 通知，切换到空态
用户 输入搜索关键词 过滤当前列表，只显示名称/描述匹配的项目 实时筛选

来源管理操作

触发 操作 系统行为 反馈
用户 点击「+ 添加目录」 打开 NSOpenPanel，选择目录 添加成功后刷新来源列表
用户 点击「+ 添加仓库」 后台执行 git clone，不阻塞 UI 来源列表显示克隆进度，完成后 Toast
用户 点击「一键更新」 后台执行 git pull 所有仓库 显示进度，完成后 Toast
系统 来源目录扫描失败 标记该来源为不可用，保留在列表 显示错误提示，可重试或删除

边界情况

场景 处理方式 反馈
目标目录无写权限 同步失败，跳过无法写入的项 Toast 提示具体失败原因
同名冲突 显示冲突预览，按策略处理 Toast 提示冲突数量
删除不存在的链接 静默忽略 无反馈
Git 克隆超时 标记为失败，保留重试入口 Toast 提示失败，可点击重试
