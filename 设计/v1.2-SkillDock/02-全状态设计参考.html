5. Toast 反馈

所有操作的成功/失败/警告反馈。

Toast 规范

类型 时长 颜色 最大数量
Success 2.0s #40c057 3
Warning 3.0s #fab005 2
Error 5.0s #fa5252 1

Toast 文案

操作 类型 文案
添加来源成功 Success 已添加来源目录
删除来源 Success 已删除来源
Git 仓库添加 Success 正在克隆仓库...
Git 同步失败 Error 同步失败：{具体原因}
删除 Skill Success 已删除 Skill
目录不存在 Warning 目录不存在，已标记

已添加来源目录
目录不存在，已标记
同步失败：目标目录已存在
交互规则

6. 交互行为规则表

触发 用户操作 系统行为 反馈
用户 点击左侧应用 Tab 切换到对应应用的 Skills 列表 高亮当前应用，更新内容区标题
用户 点击搜索框输入关键词 实时过滤 Skills（名称/描述/来源） 无结果时显示空态
用户 点击技能卡片"删除" 从应用目录删除该 Skill Toast: 已删除 Skill
用户 点击左侧"来源" 打开来源管理弹窗 显示本地目录和 Git 仓库列表
用户 点击"+ 添加目录" 打开系统目录选择器，选择后添加到来源列表 Toast: 已添加来源目录
用户 点击"+ 添加仓库" 从 Git 克隆仓库到临时目录 Toast: 正在克隆仓库... → 成功/失败
用户 点击来源项"删除" 删除该来源配置 Toast: 已删除来源
用户 点击左侧"设置" 打开设置弹窗 显示版本号和主题选项
系统 启动时 加载配置，扫描所有来源目录 显示加载态 → 正常态/错误态
系统 检测到目录不存在 标记该来源为不可用，显示错误态 Toast 警告 + UI 错误提示
系统 Git 同步失败 显示错误状态，不自动重试 Toast Error + 错误态 UI
边界情况

7. 边界情况

7.1 长文本截断

场景 处理方式
Skill 名称过长 ellipsis 截断，hover 显示完整名称
来源路径过长 ellipsis 截断
Git 仓库 URL过长 ellipsis 截断

7.2 大量数据

场景 处理方式
Skills 数量 > 100 滚动加载，网格自适应
来源数量 > 10 来源列表滚动

7.3 网络问题

场景 处理方式
Git 克隆网络超时 Toast Error: 网络超时，请重试
Git 仓库不存在 Toast Error: 仓库不存在
